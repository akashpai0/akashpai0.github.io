# akashpai0.github.io

Minimalist personal website built with Next.js and deployed to GitHub Pages.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

Static files are output to the `out/` directory.

## Deploy

1. Create a GitHub repo named `akashpai0.github.io`
2. Push this code to the `main` branch
3. In repo **Settings → Pages**, set **Source** to **GitHub Actions**
4. The site will be live at [https://akashpai0.github.io](https://akashpai0.github.io)

## Content

Edit [`src/data/profile.ts`](src/data/profile.ts) to update bio, experience, education, and links.
